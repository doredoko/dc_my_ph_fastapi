----------------------------
Notes and Restrictions
----------------------------
  • When using Terminal Service in server OS, please run Setup Utility Tools with installation mode.
    The method to run the Installation Mode is as follows.
    - Open [Add or Remove programs] in [Control Panel] and then run Setup Utility Tools.

  • During startup, the [User Account Control] screen is displayed depending on the permission of user who ran the tools. In that case, click [Allow] or enter administrator password and click [OK]. In addition, the [Program Compatibility Assistant] screen is displayed when quitting the tools. When [This program is installed correctly.] is selected, this screen will not be displayed next time.

  • Selecting NDS Queue
    Printing cannot be performed if a port/queue on the NDS tree that does not exist in the destination port is selected in the Setup Disk Creation Tool.

  • When "An error has occurred during version upgrade" is displayed during version upgrade, delete the print driver and create a new printer after restart.

  • Restarting System During Installation/Version Upgrade
    When installing the print driver or upgrading the driver version, always restart the system following the instructions given in the message displayed, before using the print driver. Unexpected errors may occur if the system is not restarted.

  • Reflecting Document Size Using Setup Tool
    When a paper unique to the driver is selected for document size, the document size will not be reflected correctly.

  • If "Current printer settings may be changed if you install the new driver. Do you want to continue?" is displayed when installing driver with the Setup Tool, select the check box "Upgrade driver using setup disk" in the Setup Disk Creation Tool, and create a setup disk.

  • If Setup Tool is created from a floppy disk using Setup Disk Creation Tool with "Upgrade driver using setup disk" selected, you cannot run the verup.exe in the floppy disk. Please run setup.exe. By running setup.exe, the print driver will also be upgraded.

  • Method to Specify Port for Network Printer
    Network printer port cannot be specified from the [Browse for Printer] screen in Setup Disk Creation Tool. Enter the port name directly in [Network path of printer] at the [Add Port] screen.

  • Reflecting Printing Settings for Setup Tool
    When you run the Setup Tool with [Allow general users to run setup] specified, the printing settings saved in the setup disk will be reflected to the printing settings of the administrator account. In addition, even when you run the Setup Tool without [Allow general users to run setup] specified but with the administrators account specified at the [User Account Control] screen, the printing settings saved in the setup disk will be reflected to the printing settings of the administrator account. To reflect the settings to the printing settings of all users including the user who ran the Setup Tool, configure [Printing Defaults] of the printer when creating the setup disk. You can set [Printing Defaults] from the [Advanced] tab of the Printer Properties dialog.
