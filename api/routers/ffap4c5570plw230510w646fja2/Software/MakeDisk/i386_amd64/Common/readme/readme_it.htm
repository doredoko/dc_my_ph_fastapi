----------------------------
Note e Restrizioni
----------------------------
  • Quando si utilizza il Servizio Terminale nel sistema operativo del server, eseguire gli Strumenti Utility di Configurazione con la modalità di installazione.
    Il metodo per eseguire la modalità di installazione è il seguente.
    - Aprire [Aggiungi o elimina programmi] nel [Pannello di controllo] ed eseguire Strumenti Utility di Configurazione.

  • Durante l'avvio, viene visualizzata la schermata [Controllo Account Utente] a seconda dell'autorizzazione dell'utente che ha eseguito gli strumenti. In tal caso, fare clic su [Consenti] o inserire la password dell'amministratore, quindi fare clic su [OK]. Inoltre, quando si esce dagli strumenti viene visualizzata la schermata [Assistente di Compatibilità Programma]. Quando viene selezionata l’opzione [Questo programma è stato installato correttamente.], questa schermata non verrà visualizzata la prossima volta.

  • Selezione della coda NDS
    Non sarà possibile eseguire la stampa se nello Strumento di creazione del disco di configurazione viene selezionata una porta/coda sull'albero NDS che non esiste nella porta di destinazione.

  • Quando in fase di aggiornamento della versione viene visualizzato il messaggio "Si è verificato un errore in fase di aggiornamento della versione", eliminare il driver di stampa e creare una nuova stampante dopo il riavvio.

  • Riavvio del sistema durante l'installazione/aggiornamento della versione
    Quando viene installato il driver di stampa o si aggiorna la versione del driver, riavviare sempre il sistema attenendosi alle istruzioni fornite nel messaggio visualizzato, prima di utilizzare il driver di stampa. Se il sistema non viene riavviato si potrebbero verificare errori imprevisti.

  • Riflettere le dimensioni del documento utilizzando lo strumento di configurazione
    Quando si seleziona una carta unica per il driver per il formato del documento, il formato del documento non viene riflesso correttamente.

  • Se viene visualizzato il messaggio “Le impostazioni attuali della stampante possono essere modificate se si installa il nuovo driver. Si desidera continuare?” quando si installa il driver con lo Strumento di Configurazione, selezionare la casella di spunta “Aggiorna il driver usando il disco di configurazione” nello Strumento di creazione del disco di configurazione, e creare un disco di configurazione.

  • Se lo Strumento di Configurazione viene creato da un floppy disk usando lo Strumento di creazione del disco di configurazione con “Aggiorna il driver usando il disco di configurazione”, non sarà possibile eseguire il file verup.exe nel floppy disk. Eseguire setup.exe. Eseguendo setup.exe, verrà aggiornato anche il driver di stampa.

  • Metodo per specificare la porta per la stampante di rete
    La porta della stampante di rete non può essere specificata dalla schermata [Sfoglia e cerca la stampante] nello Strumento di creazione del disco di configurazione. Inserire il nome della porta direttamente in [Percorso di rete della stampante] nella schermata [Aggiungi porta].

  • Riflettere le impostazioni di stampa per lo strumento di configurazione
    Quando si esegue lo Strumento di configurazione con l’opzione [Consenti agli utenti generici di eseguire la configurazione] attivata, le impostazioni di stampa salvate nel disco di configurazione si rifletteranno sulle impostazioni di stampa dell'account amministratore. Inoltre, anche quando si esegue lo Strumento di configurazione senza l’opzione [Consenti agli utenti generici di eseguire la configurazione] indicata ma con l'account dell'amministratore specificato nella schermata [Controllo account utente], le impostazioni di stampa salvate nel disco di configurazione si rifletteranno sulle impostazioni di stampa dell'account dell'amministratore. Per riflettere le impostazioni sulle impostazioni di stampa di tutti gli utenti, incluso l'utente che ha eseguito lo Strumento di Configurazione, configurare [Dettagli di stampa] della stampante quando si crea il disco di configurazione. Sarà possibile impostare i [Valori predefiniti di stampa] dalla scheda [Avanzate] della finestra di dialogo Proprietà della Stampante.
