----------------------------
注意/限制事項
----------------------------
  • 伺服器作業系統上使用終端機服務時，請以安裝模式執行。
    以安裝模式執行的方法如下：
    - 從[控制台]的[新增和移除應用程式]執行本工具。

  • 根據執行使用者的權限，啟動時可能顯示「使用者帳戶控制」畫面。此時，請按「許可」或輸入管理員密碼後按「確定」。此外，本工具結束時會顯示「程式相容性助理畫面」。若選擇「此程式已正確安裝」，此畫面將不再顯示。

  • 選擇NDS佇列
    若在設定磁碟建立工具指定不存在於目的地連接埠的NDS樹連接埠/佇列時，將無法列印。

  • 版本升級時顯示「版本升級時發生錯誤」時，請刪除印表機驅動程式。重新啟動電腦後，再建立新的印表機。

  • 安裝/版本升級時重新啟動系統
    進行印表機驅動程式的安裝或版本升級時，請依照顯示的訊息，在使用印表機驅動程式之前，務必重新啟動系統。 若不重新啟動，可能會發生無法預期的錯誤。

  • 使用使用者設定工具反映原稿尺寸
    原稿尺寸中指定了驅動程式固有的紙張時，原稿尺寸可能不會正確反映。

  • 以使用者設定工具安裝驅動程式時，若顯示「若安裝新的驅動程式，已安裝有驅動程式的印表機設定可能會有變更。是否安裝？」，請在設定磁碟建立工具勾選「使用設定磁碟進行版本升級」，建立設定磁碟。

  • 在設定磁碟建立工具勾選「使用設定磁碟進行版本升級」並在磁碟建立了使用者設定工具時，將無法執行磁碟上的verup.exe。請執行setup.exe。藉由執行setup.exe，將會升級印表機驅動程式。

  • 指定網路印表機連接埠的方法
    從建立設定磁碟工具的[瀏覽印表機]畫面無法指定網路印表機的連接埠。請在[新增印表機]畫面的[網路印表機路徑]中直接輸入連接埠名稱。

  • 有關使用者設定工具列印設定的顯示
    執行指定了[允許一般使用者執行]的使用者設定工具時，設定磁碟中儲存的列印設定，將在系統管理員帳戶的列印設定中顯示。此外，執行了不指定[允許一般使用者執行]的使用者設定工具，並在[使用者帳戶控制]畫面指定系統管理員帳戶時，設定磁碟中儲存的列印設定也會在系統管理員帳戶的列印設定中顯示。如欲使執行了使用者設定工具的使用者等在所有使用者的列印設定中顯示出來，請在建立設定磁碟時設定所指定的印表機的[列印預設值]。 可在印表機屬性的[進階]中設定[列印預設值]。
