------------------------------
Catatan dan Pembatasan
------------------------------
  • Apabila menggunakan Layanan Terminal di OS server, silakan jalankan Setup Utility Tool dengan mode penginstalan.
    Metode untuk menjalankan Mode Penginstalan adalah sebagai berikut.
    - Buka [Tambah atau Hapus program] di [Panel Kontrol] lalu jalankan Setup Utility Tool.

  • Selama penyalaan, layar [Kontrol Akun Pengguna] ditampilkan sesuai dengan izin pengguna yang menjalankan alat itu. Dalam kasus tersebut, klik [Izinkan] atau masukkan sandi administrator dan klik [OK]. Selain itu, layar [Program Asisten Kompatibilitas] ditampilkan saat keluar dari alat. Apabila [Program ini diinstal dengan benar] dipilih, layar ini tidak akan ditampilkan ke depannya.

  • Memilih Antrean NDS
    Mencetak dapat dilakukan jika yang dipilih di Setup Disk Creation Tool adalah port/antrean pada pohon NDS yang tidak ada dalam port tujuan.

  • Saat "Kesalahan terjadi saat memperbarui versi" ditampilkan saat memperbarui versi, hapus driver cetak dan buat printer baru setelah nyalakan ulang.

  • Menyalakan Ulang Sistem Selama Penginstalan/Upgrade Versi
    Ketika menginstal driver cetak atau meningkatkan versi driver, selalu nyalakan ulang sistem dengan mengikuti instruksi yang diberikan dalam pesan yang ditampilkan, sebelum menggunakan driver cetak. Kesalahan tak terduga dapat terjadi jika sistem tidak dinyalakan ulang.

  • Menampilkan Ukuran Dokumen Menggunakan Setup Tool
    Apabila yang dipilih untuk ukuran dokumen adalah kertas yang unik bagi driver, ukuran dokumen tidak akan ditampilkan secara benar.

  • Jika "Pengaturan printer saat ini dapat diubah jika Anda menginstal driver yang baru. Apa Anda ingin melanjutkan?" muncul saat menginstal driver menggunakan Setup Tool, centang kotak "Upgrade driver menggunakan disk penyetelan" di Setup Disk Creation Tool, lalu buat disk penyetelan.

  • Jika Setup Tool dibuat dari floppy disk menggunakan Setup Disk Creation Tool dengan "Upgrade driver menggunakan disk penyetelan" dicentang, Anda tidak dapat menjalankan verup.exe di floppy disk. Harap jalankan setup.exe. Dengan menjalankan setup.exe, driver cetak juga akan ditingkatkan.

  • Metode untuk Menentukan Port bagi Printer Jaringan
    Port printer jaringan tidak dapat ditentukan dari layar [Telusuri Printer] pada Setup Disk Creation Tool. Masukkan nama port secara langsung di [Jalur jaringan printer] di layar [Tambah Port].

  • Menampilkan Pengaturan Cetak untuk Setup Tool
    Saat Anda menjalankan Setup Tool dengan memilih [Izinkan pengguna umum untuk menjalankan penyetelan], pengaturan pencetakan yang disimpan dalam disk penyetelan akan ditampilkan ke pengaturan pencetakan dari akun administrator. Selain itu, meskipun saat Anda menjalankan Setup Tool tanpa memilih [Izinkan pengguna umum untuk menjalankan penyetelan] namun tanpa menentukan akun administrator di layar [Kontrol Akun Pengguna], pengaturan cetak yang disimpan di disk penyetelan akan ditampilkan ke pengaturan pencetakan akun administrator. Untuk menampilkan pengaturan ke pengaturan pencetakan semua pengguna, termasuk pengguna yang menjalankan Setup Tool, konfigurasikan [Default Pencetakan] printer saat membuat disk penyetelan. Anda dapat menyetel [Default Pencetakan] dari tab [Tingkat Lanjut] dialog Properti Printer.
