----------------------------
หมายเหตุและข้อจำกัด
----------------------------
  • เมื่อใช้บริการเทอร์มินัลใน OS เซิร์ฟเวอร์ของ ให้ใช้งาน Setup Utility Tools ด้วยโหมดติดตั้ง
    วิธีใช้งานโหมดติดตั้งมีดังนี้
    - เปิด [เพิ่มหรือเอาโปรแกรมออก] ใน [แผงควบคุม] และเริ่มดำเนินการ Setup Utility Tools

  • ในระหว่างการเริ่มต้น หน้าจอ [ควบคุมบัญชีผู้ใช้] อาจแสดงขึ้น ทั้งนี้ขึ้นอยู่กับสิทธิ์ของผู้ใช้ที่ใช้งานเครื่องมือ ในกรณีนี้ ให้คลิก [อนุญาต] หรือใส่รหัสผ่านของผู้ดูแลระบบ และคลิกปุ่ม [ตกลง] นอกจากนี้ หน้าจอ [ตัวช่วยเกี่ยวกับความเข้ากันได้ของโปรแกรม] อาจปรากฏขึ้นเมื่อคุณออกจากการใช้งานเครื่องมือ แต่หากคุณเลือก [ติดตั้งโปรแกรมนี้อย่างถูกต้อง] แล้ว หน้าจอนี้จะไม่ปรากฏขึ้นอีกในครั้งต่อไป

  • การเลือกคิว NDS
    คุณจะไม่สามารถดำเนินการพิมพ์ได้หากเลือก พอร์ต/คิว ใน NDS ทรีที่ไม่ได้อยู่ในพอร์ตปลายทางที่เลือกไว้ใน Setup Disk Creation Tool

  • เมื่อมีข้อความ "มีข้อผิดพลาดเกิดขึ้นในระหว่างการอัพเกรดเวอร์ชั่น" ปรากฏขึ้นในระหว่างการอัพเกรดเวอร์ชั่น ให้ลบโปรแกรมควบคุมการพิมพ์ และสร้างเครื่องพิมพ์ใหม่หลังจากที่ได้รีสตาร์ทแล้ว

  • การรีสตาร์ทระบบใหม่ในระหว่างการติดตั้ง / การอัพเกรดเวอร์ชั่น
    เมื่อทำการติดตั้งโปรแกรมควบคุมการพิมพ์หรืออัพเกรดเวอร์ชั่นของโปรแกรมควบคุม ให้รีสตาร์ทระบบตามคำแนะนำที่ระบุไว้ในข้อความที่ปรากฏขึ้นก่อนเริ่มใช้โปรแกรมควบคุมเสมอเพราะอาจมีข้อผิดพลาดที่ไม่คาดคิดเกิดขึ้นได้หากไม่ทำการรีสตาร์ทระบบก่อน

  • การแสดงขนาดเอกสารโดยใช้ Setup Tool
    เมื่อมีการเลือกกระดาษเฉพาะสำหรับโปรแกรมควบคุมไว้ ขนาดเอกสารจะไม่สามารถแสดงให้เห็นได้อย่างถูกต้อง

  • เมื่อมีข้อความ "การตั้งค่าของเครื่องพิมพ์ปัจจุบันอาจเปลี่ยนไปหากคุณติดตั้งโปรแกรมควบคุม คุณต้องการดำเนินการต่อหรือไม่" ปรากฏขึ้นเมื่อทำการติดตั้งโปรแกรมควบคุมโดยใช้ Setup Tool ให้เลือกกล่องกาเครื่องหมาย "อัพเกรดโปรแกรมควบคุมโดยใช้ดิสก์ติดตั้ง" ใน Setup Disk Creation Tool และให้ทำการสร้างดิสก์ติดตั้ง

  • หากคุณสร้าง Setup Tool จากฟลอปปี้ดิสก์โดยการใช้ Setup Disk Creation Tool ที่เลือก "อัพเกรดโปรแกรมควบคุมโดยใช้ดิสก์ติดตั้ง" ไว้ คุณจะไม่สามารถเปิดใช้งานไฟล์ verup.exe ในฟลอปปี้ดิสก์ได้ ให้เปิดใช้งานไฟล์ setup.exe แทน ซึ่งการเปิดใช้งานไฟล์ setup.exe นี้จะทำให้โปรแกรมควบคุมการพิมพ์ถูกอัพเกรดไปด้วยเช่นกัน

  • วิธีการกำหนดพอร์ตสำหรับเครื่องพิมพ์เครือข่าย
    คุณไม่สามารถระบุพอร์ตเครื่องพิมพ์เครือข่ายได้จากหน้าจอ [เรียกดูเครื่องพิมพ์] ใน Setup Disk Creation Tool โปรดระบุชื่อพอร์ตลงใน [เส้นทางเครือข่ายของเครื่องพิมพ์] ที่หน้าจอ [เพิ่มพอร์ต] โดยตรง

  • การนำการตั้งค่าการพิมพ์ไปใช้สำหรับ Setup Tool
    เมื่อคุณใช้ Setup Tool ที่เลือก [ให้ผู้ใช้ทั่วไปใช้โปรแกรมติดตั้งได้] การตั้งค่าการพิมพ์ที่บันทึกไว้ในดิสก์ติดตั้งจะถูกนำไปใช้ในการตั้งค่าการพิมพ์สำหรับบัญชีผู้ดูแลระบบ นอกจากนี้แม้เมื่อเริ่มใช้งาน Setup Tool โดยไม่ได้ระบุตัวเลือก [ให้ผู้ใช้ทั่วไปใช้โปรแกรมติดตั้งได้] แต่ด้วยสิทธิ์ของบัญชีผู้ดูแลระบบที่ระบุไว้ในหน้าจอ [ควบคุมบัญชีผู้ใช้] การตั้งค่าการพิมพ์ที่บันทึกไว้ในดิสก์ติดตั้งก็จะถูกนำไปใช้กับการตั้งค่าการพิมพ์สำหรับบัญชีผู้ดูแลระบบ หากต้องการนำการตั้งค่าไปใช้กับการตั้งค่าการพิมพ์สำหรับผู้ใช้ทั้งหมด รวมถึงผู้ใช้ที่ใช้ Setup Tool นั้น ให้กำหนดค่า [ค่าเริ่มต้นการพิมพ์] ของเครื่องพิมพ์เมื่อสร้างดิสก์ติดตั้ง คุณสามารถตั้ง [ค่าเริ่มต้นการพิมพ์] ได้จากแท็บ [ขั้นสูง] ของกล่องโต้ตอบคุณสมบัติเครื่องพิมพ์


    • Microsoft, Windows และ Windows Server เป็นเครื่องหมายการค้าจดทะเบียน หรือเครื่องหมายการค้าของ Microsoft Corporation, U.S.A. ในประเทศสหรัฐอเมริกาและประเทศอื่นๆ
