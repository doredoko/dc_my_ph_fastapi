----------------------------
注意事项/限制事项
----------------------------
  • 的服务器操作系统中使用终端服务时，请在安装模式下执行。
    在安装模式下执行的方法如下。
    - 从[控制面板]的[添加/删除程序]执行本工具。

  • 根据执行用户的权限，启动时可能会显示[用户帐户控制]画面。在此情况下，请点击[允许]，或输入管理员密码后再点击[确定]。此外，退出本工具时，将显示[程序兼容性助手]画面。若选择[这个程序已经正确安装]，则以后不再显示此画面。

  • 关于NDS队列的选择
    使用启动盘制作工具指定不存在于输出目标端口中的NDS树上的端口/队列时，将无法打印。

  • 版本升级时，如果显示“版本升级过程中发生了错误”，请删除打印机驱动程序，然后重新启动并创建新的打印机。

  • 关于安装/版本升级时系统的重新启动
    安装打印机驱动程序或者进行版本升级时，请依照显示的信息，在使用打印机驱动程序之前，务必重新启动系统。若不重新启动，可能会发生无法预期的错误。

  • 关于使用用户启动工具时原稿尺寸的反映
    原稿尺寸中指定了驱动程序固有的纸张时，原稿尺寸将不会正确反映。

  • 使用用户启动工具安装驱动程序时，如果显示“若安装新的驱动程序，已安装的打印机设定可能会变更。是否继续安装？”的信息，请在启动盘制作工具中勾选“使用启动盘进行版本升级”复选框，以创建启动盘。

  • 在启动盘制作工具中勾选“使用启动盘进行版本升级”复选框，并在软盘中创建用户启动工具时，无法执行软盘上的verup.exe。请执行setup.exe。通过执行setup.exe，打印机驱动程序的版本将被升级。

  • 关于网络打印机端口的指定方法
    使用启动盘制作工具无法从[浏览打印机]画面指定网络打印机的端口。请在[添加端口]画面的[打印机的网络路径]中直接输入端口名。

  • 关于使用用户启动工具时打印设定的反映
    已执行指定了[允许一般用户执行]的用户启动工具时，保存在启动盘中的打印设定将被反映到管理员帐户的打印设定中。此外，已执行未指定[允许一般用户执行]的用户启动工具，并在[用户帐户控制]画面指定了管理员帐户时，保存在启动盘中的打印设定也将被反映到管理员帐户的打印设定中。若要反映到执行了用户启动工具的用户等所有用户的打印设定中，请在创建启动盘时设定要指定的打印机的[打印默认值]。[打印默认值]可在打印机属性的[高级]画面中进行设定。
