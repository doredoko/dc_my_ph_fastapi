----------------------------
Observações e restrições
----------------------------
  • Ao usar o Serviço de Terminal no SO de um servidor, execute as ferramentas de configuração do utilitário no modo de instalação.
    O método para executar o modo de instalação é o seguinte.
    - Abrir [Adicionar ou remover programas] no [Painel de controle] e depois executar Ferramentas de configuração de utilitários.

  • Durante a inicialização, a tela [Controle de conta de usuário] é exibida dependendo da permissão do usuário que executa as ferramentas. Nesse caso, Clique em [Permitir] ou insira a senha de administrador e Clique em [OK]. Além disso, a tela [Assistente de compatibilidade de programa] é exibida ao encerrar as ferramentas. Ao selecionar [Este programa foi instalado corretamente.], essa tela não será exibida na próxima vez.

  • Selecionando fila NDS
    A impressão não pode ser feita se uma porta/fila na árvore de NDS que não existe na porta de destino for selecionada na ferramenta de criação de disco de instalação.

  • Quando aparecer "Ocorreu um erro durante atualização da versão" durante a atualização da versão, excluir o driver de impressão e criar uma nova impressora após reiniciar.

  • Reiniciar sistema durante instalação/atualização de versão
    Ao instalar o driver de impressão ou atualizar a versão do driver, sempre reiniciar o sistema seguindo as instruções fornecidas na mensagem exibida, antes de usar o driver de impressão. É possível que ocorram erros inesperados se o sistema não for reiniciado.

  • Refletir tamanho de documento ao usar ferramenta de instalação
    Quando selecionado um papel exclusivo para o driver para tamanho do documento, esse tamanho não será refletido corretamente.

  • Se "As configurações da impressora atual poderão mudar se o novo driver for instalado. Deseja continuar?" for exibido ao instalar driver com a ferramenta de instalação, selecione a caixa de verificação "Atualizar driver usando disco de instalação" na ferramenta de criação de disco de instalação, e crie um disco de instalação.

  • Se a ferramenta de instalação for criada a partir de um disquete usando a ferramenta de criação de disco de instalação com "Atualizar driver usando disco de instalação" selecionado, não é possível executar verup.exe no disquete. Execute setup.exe. Ao executar setup.exe, o driver de impressão também será atualizado.

  • Método para especificar porta para impressora de rede
    A porta da impressora de rede não pode ser especificada a partir da tela [Procurar impressora] na ferramenta de criação de disco de instalação. Inserir o nome da porta diretamente em [Caminho de rede da impressora] na tela [Adicionar porta].

  • Refletir configurações de impressão para ferramenta de instalação
    Ao executar a Ferramenta de instalação com [Permitir que usuários gerais executem a configuração] especificado, as configurações de impressão salvas no disco de instalação serão refletidas nas configurações de impressão da conta do administrador. Além disso, mesmo quando executar a Ferramenta de instalação sem especificar [Permitir que usuários gerais executem a configuração], mas com a conta de administradores especificada na tela [Controle de conta de usuário], as configurações de impressão salvas no disco de instalação serão refletidas nas configurações de impressão da conta de administrador. Para refletir as configurações nas configurações de impressão de todos os usuários incluindo o usuário que executou a Ferramenta de instalação, configurar os [Padrões de impressão] da impressora quando criar o disco de instalação. É possível definir os [Padrões de impressão] a partir da guia [Avançado] da caixa de diálogo Propriedades da impressora.
