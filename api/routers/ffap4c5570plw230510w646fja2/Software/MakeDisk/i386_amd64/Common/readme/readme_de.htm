----------------------------
Hinweise und Beschränkungen
----------------------------
  • Wenn Sie Terminal Service im Server-Betriebssystem verwenden, führen Sie bitte Setup Utility Tools im Installationsmodus aus.
    Die Methode, den Installationsmodus zu aktivieren, ist wie folgt.
    - Öffnen [Programme hinzufügen oder entfernen] in [Bedienfeld] und dann Setup Utility Tools ausführen.

  • Während des Startvorgangs wird der Bildschirm [Benutzerkontosteuerung], abhängig von der Erlaubnis des Benutzers, der die Tools ausgeführt hat, angezeigt. Klicken Sie in diesem Fall auf [Erlauben] oder geben Sie das Administratorpasswort ein und klicken Sie auf [OK]. Zusätzlich dazu wird der Bildschirm [Programmkompatibilitätsassistent] angezeigt, wenn die Tools beendet werden. Wenn [Dieses Programm wurden richtig installiert.] ausgewählt wird, wird dieser Bildschirm das nächste Mal nicht mehr angezeigt.

  • NDS-Warteschlange auswählen
    Druck kann nicht durchgeführt werden, falls ein Port/Warteschlange auf dem NDS-Baum, der nicht am Ziel-Port existiert, im Setup Disk Creation Tool ausgewählt wird.

  • Wenn „Ein Fehler ist während der Aktualisierung der Version aufgetreten“ angezeigt wird, löschen Sie den Druckertreiber und erstellen Sie einen neuen Drucker nach dem Neustart.

  • Neustart des Systems während der Installation/Aktualisierung der Version
    Starten Sie während der Installation des Druckertreibers oder der Aktualisierung der Treiberversion immer das System, nachdem Sie die Anweisungen erhalten haben, die in der angezeigten Mitteilung angegeben werden, bevor Sie den Druckertreiber verwenden. Unerwartete Fehler können auftreten, falls das System nicht neu gestartet wird.

  • Berücksichtigung der Dokumentengröße mittels Setup Tool
    Wenn ein für den Treiber spezielles Papier für die Dokumentengröße ausgewählt wird, wird die Dokumentengröße nicht richtig berücksichtigt.

  • Falls „Aktuelle Druckertreibereinstellungen könnten geändert werden, wenn sie den neuen Treiber installieren. Möchten Sie fortfahren?“ bei der Installation des Treibers mit dem Einrichtungswerkzeug angezeigt wird, wählen Sie das Kontrollkästchen „Treiber über Einrichtungsdiskette aktualisieren“ im Setup Disk Creation Tool aus und erstellen Sie eine Einrichtungsdiskette.

  • Wenn das Einrichtungswerkzeug über eine Diskette mittels Setup Disk Creation Tool mit „Treiber mittels Einrichtungsdiskette erstellen“ erstellt wird, können Sie verup.exe auf der Diskette nicht ausführen. Bitte führen Sie setup.exe aus. Wenn Sie setup.exe ausführen, wird der Druckertreiber auch aktualisiert.

  • Methode zur Festlegung des Ports für den Netzwerkdrucker
    Der Netzwerkdrucker-Port kann nicht über den Bildschirm [Nach Drucker suchen] im Setup Disk Creation Tool angegeben werden. Geben Sie den Port-Namen direkt in [Netzwerkpfad des Druckers] im Bildschirm [Port hinzufügen] ein.

  • Berücksichtigung Druckereinstellungen für Einrichtungswerkzeug
    Wenn Sie das Einrichtungswerkzeug mit [Allgemeinen Benutzern erlauben, die Einrichtung auszuführen] ausführen, werden die auf der Einrichtungsdiskette gespeicherten Druckereinstellungen in den Druckereinstellungen des Administratorkontos berücksichtigt. Zusätzlich dazu und obwohl Sie das Einrichtungswerkzeug mit [Allgemeinen Benutzern erlauben, die Einrichtung auszuführen], aber mit den im Bildschirm [Benutzerkontosteuerung] angegeben Administratorkonten ausführen, werden die auf der Einrichtungsdiskette gespeicherten Druckereinstellungen in den Druckereinstellungen des Administratorkontos berücksichtigt. Um die Einstellungen in den Druckereinstellungen aller Benutzer wiederzugeben, einschließlich dem Benutzer, der das Einrichtungswerkzeug ausgeführt hat, konfigurieren Sie [Druckerstandardeinstellungen] des Druckers, wenn Sie die Einrichtungsdiskette erstellen. Sie können die [Druckerstandardeinstellungen] über die Registerkarte [Erweitert] des Dialogfeldes Druckereigenschaften einstellen.
