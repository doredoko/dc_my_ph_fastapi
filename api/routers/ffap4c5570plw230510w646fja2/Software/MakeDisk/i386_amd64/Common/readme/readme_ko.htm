----------------------------
주의 및 제한 사항
----------------------------
  • 서버 OS에서 터미널 서비스를 사용하려면 설치 모드에서 실행하십시오.
    설치 모드에서 실행하는 방법은 다음과 같습니다.
    - [제어판]의 [프로그램 추가/제거]에서 이 툴을 실행합니다.

  • 컴퓨터를 시작할 때 실행한 사용자 권한에 따라 "사용자 계정 컨트롤" 화면이 표시될 수 있습니다. 이 경우 "허가"를 클릭하거나 관리자 비밀번호를 입력하고 "확인"을 클릭하십시오. 또한 이 툴을 종료할 때 "프로그램 호환성 관리자"가 표시됩니다. "이 프로그램이 제대로 설치되었습니다"를 선택하면 다음 화면이 표시됩니다.

  • NDS 큐 선택
    설정 디스크 작성 툴에서 대상 포트에 존재하지 않은 NDS 트리상의 포트/큐를 지정하면 프린트할 수 없습니다.

  • 버전 업그레이드 시 "버전 업그레이드 중에 오류가 발생했습니다"라고 표시되면 프린터 드라이버를 삭제하고, 다시 시작한 후 새 프린터를 작성하십시오.

  • 설치/버전 업그레이드 시 시스템 다시 시작
    프린터 드라이버 설치 또는 버전 업그레이드를 실행한 경우, 표시되는 메시지에 따라 프린터 드라이버를 사용하기 전에 반드시 시스템을 다시 시작하십시오. 다시 시작하지 않으면 예기치 않은 오류가 발생하는 경우가 있습니다.

  • 사용자 설정 툴의 원고 크기 반영
    원고 크기에 드라이버 고유의 용지가 지정되면 원고 크기가 올바르게 반영되지 않는 경우가 있습니다.

  • 사용자 설정 툴에서 드라이버 설치 시 "새 드라이버를 설치할 경우 이미 설치되어 있는 프린터 설정이 변경될 수 있습니다. 설치하시겠습니까?" 라고 표시된 경우 설치 디스크 작성 툴에서 "설정 디스크에서 버전 업그레이드를 실시" 확인란을 선택하고 설정 디스크를 작성하십시오.

  • 설치 디스크 작성 툴에서 "설정 디스크에서 버전 업그레이드를 실시" 확인란을 선택하고 플로피 디스크에 사용자 설정 툴을 작성한 경우, 플로피 디스크상의 verup.exe는 실행할 수 없습니다. setup.exe를 실행하십시오. setup.exe를 실행하면 프린터 드라이버가 버전 업그레이드됩니다.

  • 네트워크 프린터 포트 지정 방법
    설치 디스크 작성 툴의 [프린터 찾아보기] 화면에서 네트워크 프린터의 포트를 지정할 수 없습니다. [포트 추가] 화면의 [프린터 네트워크 경로]에 포트 이름을 입력하십시오.

  • 사용자 설정 툴에서 인쇄 설정 반영
    [일반 사용자에게 설치 실행 허가]를 지정한 사용자 설정 툴을 실행한 경우, 설치 디스크에 저장된 인쇄 설정이 관리자 계정의 인쇄 설정에 반영됩니다. 또한 [일반 사용자에게 설치 실행 허가]를 지정하지 않은 사용자 설정 툴을 실행하고 [사용자 계정 컨트롤] 화면에서 관리자 계정을 지정한 경우에도 설치 디스크에 저장된 인쇄 설정이 관리자 계정의 인쇄 설정에 반영됩니다. 사용자 설정 툴을 실행한 사용자 등, 모든 사용자의 인쇄 설정에 반영하려면 설치 디스크 작성 시 지정할 프린터의 [기본값으로 인쇄]를 설정하십시오. [기본값으로 인쇄]는 프린터 속성의 [고급]에서 설정합니다.
