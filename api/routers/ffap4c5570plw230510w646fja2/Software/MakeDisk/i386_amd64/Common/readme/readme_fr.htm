----------------------------
Remarques et Restrictions
----------------------------
  • Lorsque vous utilisez le Service Terminal dans le système d'exploitation serveur, veuillez exécuter les outils utilitaires de configuration avec le mode d'installation.
    La méthode pour exécuter le mode d'installation est la suivante.
    - Ouvrez [Ajouter ou supprimer des programmes] dans le [Panneau de commande], puis exécutez les outils utilitaires de configuration.

  • Au démarrage, l'écran [Contrôle de compte utilisateur] s'affiche en fonction de l'autorisation de l'utilisateur qui a exécuté les outils. Dans ce cas, cliquez sur [Autoriser] ou saisissez le mot de passe administrateur et cliquez sur [OK]. De plus, l'écran [Assistant de compatibilité des programmes] s'affiche lorsque vous quittez les outils. Lorsque [Ce programme est correctement installé.] est sélectionné, cet écran ne s'affichera pas la prochaine fois.

  • Sélection de la file d'attente NDS
    L'impression ne peut pas être effectuée si un port/une file d'attente dans l'arborescence NDS qui n'existe pas dans le port de destination est sélectionné dans l'outil de création de disque d'installation.

  • Lorsque « Une erreur s'est produite lors de la mise à niveau de la version » s'affiche pendant la mise à niveau de la version, supprimez le pilote d'impression et créez une nouvelle imprimante après le redémarrage.

  • Redémarrage du système pendant l'installation/la mise à niveau de la version
    Lors de l'installation du pilote d'impression ou de la mise à niveau de la version du pilote, redémarrez toujours le système en suivant les instructions données dans le message affiché avant d'utiliser le pilote d'impression. Des erreurs inattendues peuvent se produire si le système n'est pas redémarré.

  • Montrant le format du document à l’aide de l’outil de configuration
    Lorsqu'un papier propre au pilote est sélectionné pour le format du document, le format du document ne sera pas correctement montré.

  • Si « Les paramètres actuels de l'imprimante peuvent être modifiés si vous installez le nouveau pilote. Voulez-vous continuer ?v» s'affiche lors de l'installation du pilote avec l'outil de configuration, cochez la case « Mettre à niveau le pilote à l'aide du disque d'installation » dans l'outil de création de disque d'installation et créez un disque d'installation.

  • Si l'outil de configuration est créé à partir d'une disquette à l'aide de l'outil de création de disque d'installation avec « Mettre à niveau le pilote à l'aide du disque d'installation » sélectionné, vous ne pouvez pas exécuter le verup.exe sur la disquette. Veuillez exécuter setup.exe. En exécutant setup.exe, le pilote d'impression sera également mis à niveau.

  • Méthode de désignation du port pour l'imprimante réseau
    Le port d'imprimante réseau ne peut pas être désigné à partir de l'écran [Rechercher une imprimante] dans l'outil de création de disque d'installation. Saisissez directement le nom du port dans [Chemin réseau de l'imprimante] sur l'écran [Ajouter un port].

  • Montrant les paramètres d'impression pour l'outil de configuration
    Lorsque vous exécutez l'outil de configuration avec [Autoriser les utilisateurs généraux à exécuter le programme d’installation] spécifié, les paramètres d'impression enregistrés sur le disque d’installation seront montrés dans les paramètres d'impression du compte administrateur En outre, même lorsque vous exécutez l'outil de configuration sans [Autoriser les utilisateurs généraux à exécuter le programme d’installation] spécifié mais avec le compte administrateur spécifié sur l'écran [Contrôle de compte utilisateur], les paramètres d'impression enregistrés sur le disque d’installation seront montrés dans les paramètres d'impression. du compte administrateur. Pour montrer les paramètres sur les paramètres d'impression de tous les utilisateurs, y compris l'utilisateur qui a exécuté l'outil de configuration, configurez [Paramètres d'impression par défaut] de l'imprimante lors de la création du disque d’installation. Vous pouvez définir les [Paramètres d'impression par défaut] dans l'onglet [Avancé] de la boîte de dialogue Propriétés de l'imprimante.
